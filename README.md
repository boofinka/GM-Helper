# GM-Helper

A tool to help RPG game masters describe scenes, characters, items and more.

## Run the web app (development)

Prerequisites: Node.js (16+) and npm.

Install dependencies and start the Vite dev server:

```powershell
cd "PATH/TO/CODE"
npm install
npm run dev
```

Open the dev URL shown in the terminal (e.g. `http://localhost:5174/`).

The web app loads JSON content from `src_jsons/` and uses `app.js` for template generation.
